# 【论文阅读】Improving Multi-modal Recommender Systems by Denoising and Aligning Multi-modal Content and User Feedback

通过去噪和对齐多模态内容和用户反馈来改进多模态推荐系统

关键词：Multi-modal Recommender System, Noisy Multi-modal Content, Noisy User Feedback, Aligning Multi-modal Content and User Feedback

## 摘要

多模式推荐系统（MRS）在各种在线网络平台中至关重要，近年来引起了相当大的关注。

问题：

1）嘈杂的多模态内容的挑战，(2) 嘈杂的用户反馈，以及 (3) 调整多模式内容和用户反馈。

方法：

为了减轻多模态内容中的噪声，DAMRS 首先构建由跨模态的一致内容相似性确定的项-项图。

为了对用户反馈进行去噪，DAMRS 将观察到的反馈的概率与多模态内容相关联，并设计了去噪 BPR 损失。

此外，DAMRS 实现了由用户偏好引导的对齐，以增强特定于任务的项目表示，以及由分级项目关系引导的对齐，以提供更细粒度的对齐。
