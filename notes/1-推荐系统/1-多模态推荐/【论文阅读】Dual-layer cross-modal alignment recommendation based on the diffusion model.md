# 【论文阅读】Dual-layer cross-modal alignment recommendation based on the diffusion model

标题: 基于扩散模型的双层跨模态对齐推荐。

关键词：

Multimodal recommendation 、Cross-modal alignment 、Imbalanced data 、Self-supervised learning Graph convolution network



## 摘要

问题：多模态推荐现有的方法未能充分挖掘各模态的信息，在对齐过程中引入语义偏差，导致用户行为信息丢失，模态关联和用户偏好难以捕捉。

解决：提出一种基于扩散模型的双层跨模态对齐推荐方法( DCAR-DM )。具体来说，该方法使用扩散模型为每个模态生成具有相同交互次数的用户-项目交互图，然后通过模态贡献调节机制为每个模态筛选最符合用户兴趣的交互来重构该交互图。然后，DCAR - DM通过图卷积网络( GCN )聚合模态信息生成最终的用户和项目嵌入。此外，DCAR - DM使用双层跨模态对齐机制来指导模态对齐。

特征对齐层将模态表示为高斯分布，并最小化均值和标准差来对齐模态间的特征。行为对齐层通过对比学习利用真实的用户-项目交互行为，进一步对齐模态特征，以纠正特征对齐过程中产生的语义偏差。



## 方法

### Multimodal diffusion model

输入：原始的用户–物品二部图 $\mathcal A$（来自点击/观看等行为）。

目标：为每个模态（图中粉/黄/绿圈，视觉/文本/声学）各自生成**等量**的候选交互，再从中挑出“高质量、符合用户该模态偏好”的边，得到各模态的交互图 $\mathcal A^m$。

- **DDPM 训练**：前向加噪把 $\mathcal A$ 映到噪声空间，反向去噪逐步重建；优化采用**VLB**（变分下界），最后可写成 MSE 形式来拟合反向过程（图中“Forward/Reverse, VLB Loss”那条虚线）。

- **等量候选**：在反向去噪过程中，**为每个用户在每个模态生成相同数量的候选交互**，记作 $k_{\text{total}}$，构成候选图 $\hat{\mathcal A}^m$。这样先从数量上消除模态不平衡（冷/热用户差异不会放大）。

- **MCRM（模态贡献调节）**：对 $\hat{\mathcal A}^m$ 做“二次筛选”，依据**有效样本数**与**特征信息熵**给模态打分，只保留更能代表用户该模态偏好的边，得到最终的 $\mathcal A^m$。

- 从消融与文字描述看，MCRM 的加入能显著提升表现，说明“数量补齐后再按质量筛边”的管道很关键



###  Multimodal graph aggregation

**输入**：原始图 $\mathcal A$ 与各模态图 $\mathcal A^m$，以及每个物品的多模态特征嵌入 $E_m$。
**做法**：在这些图上用 GCN 做邻域聚合，融合来自视觉/文本/声学的关系与特征，得到用户/物品的**融合表示**（图右下“GCN”块）。核心目的：把“关系补齐+筛优”的结果真正注入到表征里。

### Dual-layer cross-modal alignment

**输入**：来自 (2) 的各模态表示与融合表示。

**动机**：仅做“特征对齐”的方法容易产生语义偏差；单靠对比学习又可能难以覆盖多模态分布。作者因此把**特征对齐**与**行为对齐**叠在一起，用互补信号纠偏。

- **Feature alignment（特征对齐）**：在模态内/模态间让表示分布靠拢（图左下“feature alignment”，文中对应 3.4.1 小节），降低模态间表示失配。目录也明确将其作为一层。

- **Behavior alignment（行为对齐）**：以融合后的用户/物品表示为**锚**，和各模态表示做**InfoNCE 对比学习**（图中“behavior alignment—InfoNCE”），让“被用户真实行为证实的语义”去拉齐各模态。目录 3.4.2 标注为第二层。

- **总对齐损失**：图底部把特征对齐损失 $L_{fe}$ 与行为对齐损失 $L_{be}$ 汇总成 $L_{align}$，再与推荐主损失一起优化（详见“Optimization process”小节）



> 先用**扩散**从原始交互图合成“每个模态等量”的候选边，再用 **MCRM** 只留下**信息量更高**的那部分，得到模态专属图 $\mathcal A^m$；接着在 $\mathcal A$ 与 $\mathcal A^m$ 上用 **GCN** 融合成用户/物品表征；最后用**“特征分布靠拢 + 行为对比拉齐”**的**双层对齐**，把各模态的语义真正对齐到用户行为上，从而提升推荐效果。
