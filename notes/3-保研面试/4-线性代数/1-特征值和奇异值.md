## 3. 奇异值：描述矩阵对空间的“拉伸强度”

奇异值比特征值更一般。

特征值主要针对 **方阵**，而奇异值可以用于任意矩阵，包括非方阵。

对于一个矩阵 ![img](https://cdn.nlark.com/yuque/__latex/de951302f41d4707b9d80ca1af34dd0f.svg)，它的奇异值来自下面这个分解：

![img](https://cdn.nlark.com/yuque/__latex/d6574fe430762d9dd64bebc955b1e8d5.svg)

这叫做 **奇异值分解**，简称 **SVD**。

其中：

- ![img](https://cdn.nlark.com/yuque/__latex/61730156a4ee0ac4b11b62f6440f5779.svg) 是左奇异向量矩阵；
- ![img](https://cdn.nlark.com/yuque/__latex/9f493997c33913987175caf4a4849955.svg) 是右奇异向量矩阵；
- ![img](https://cdn.nlark.com/yuque/__latex/a99c3dabbb2eeee45801b2b7d343cc65.svg) 是一个对角矩阵；
- ![img](https://cdn.nlark.com/yuque/__latex/a99c3dabbb2eeee45801b2b7d343cc65.svg) 对角线上的非负数就是矩阵 ![img](https://cdn.nlark.com/yuque/__latex/de951302f41d4707b9d80ca1af34dd0f.svg) 的 **奇异值**。

可以理解为：矩阵 ![img](https://cdn.nlark.com/yuque/__latex/de951302f41d4707b9d80ca1af34dd0f.svg) 对空间做变换时，会先旋转，再沿某些正交方向拉伸，最后再旋转。奇异值就是这些方向上的拉伸倍数。

## 5. 二者的核心区别

| 对比点       | 特征值                                                       | 奇异值                                                       |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 适用对象     | 主要用于方阵                                                 | 任意矩阵都可以                                               |
| 是否可能为负 | 可以为负，也可以是复数                                       | 一定是非负实数                                               |
| 几何含义     | 某些方向上方向不变，只改变长度                               | 矩阵整体变换中的主要拉伸强度                                 |
| 来源         | 解 ![img](https://cdn.nlark.com/yuque/__latex/5fe7e4bc17003eea838536a2d0448c1e.svg) | 来自 ![img](https://cdn.nlark.com/yuque/__latex/d6574fe430762d9dd64bebc955b1e8d5.svg) 或 ![img](https://cdn.nlark.com/yuque/__latex/dd8ef5e6cdc4ee13855bc4cfbee8c3e4.svg) 的特征值 |
| 稳定性       | 对一般矩阵可能较复杂                                         | 通常更稳定、更通用                                           |

## 6. 一个直观类比

可以把矩阵看成一个“空间变形器”。

**特征值**关注的是：

有没有一些特殊方向，经过矩阵变换后方向不变，只是长度变了？

**奇异值**关注的是：

这个矩阵整体上把空间最多拉长多少、压缩多少？

所以，特征值更强调“方向不变的特殊向量”，奇异值更强调“空间变换的拉伸强度”。

在机器学习、推荐系统、降维、矩阵分解中，奇异值特别常见。

这也是 PCA、LSA、矩阵补全、协同过滤等方法背后的重要数学基础。

## 8. 简单总结

特征值看的是 **方阵在某些特殊方向上的缩放倍数**；奇异值看的是 **任意矩阵对空间整体的拉伸强度**。

一句话区分：

特征值回答“哪些方向经过变换后方向不变”，奇异值回答“这个矩阵在各个主要方向上拉伸了多少”。
