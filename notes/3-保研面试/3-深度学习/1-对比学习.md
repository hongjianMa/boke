## 监督学习和自监督学习的区别

**监督学习的任务：做“有答案的练习题”。**

比如老师给你很多题：

| 输入          | 标准答案 |
| ------------- | -------- |
| 一张猫的图片  | 猫       |
| 一张狗的图片  | 狗       |
| 用户A + 商品B | 会点击   |
| 用户A + 商品C | 不会点击 |

**监督学习是在学“输入和人工标签之间的对应关系”。**

------

**自监督学习的任务：自己把数据改造成练习题。**

它没有人工给的标准答案，但可以从数据本身制造答案。

比如：

| 原始数据                   | 自己设计的任务                                   |
| -------------------------- | ------------------------------------------------ |
| 一句话：“我今天去学校上课” | 遮住“学校”，让模型猜被遮住的词                   |
| 一张图片                   | 遮住一块，让模型猜缺失区域                       |
| 一张用户-商品交互图        | 去掉一部分边，让模型预测哪些用户和商品本来有关系 |
| 一个商品的图片和文本       | 让模型判断这张图和这段文本是不是同一个商品       |

## 对比学习

**让模型学会：相似的东西靠近，不相似的东西远离。**

它不一定需要人工标签，而是自己构造“正样本”和“负样本”。

比如有一个商品 A：

- 商品 A 的图片表示：$z_i$
- 商品 A 的文本表示：$z_i^+$
- 其他商品 B、C、D 的表示：$z_1^-, z_2^-, z_3^-$

那么模型要做的事情就是：

**让商品 A 的图片表示和商品 A 的文本表示更接近；让商品 A 和其他商品的表示更远。**

常见的对比学习公式是 InfoNCE：
$$
\mathcal{L}_i =
-\log
\frac{
\exp(\mathrm{sim}(z_i, z_i^+)/\tau)
}{
\exp(\mathrm{sim}(z_i, z_i^+)/\tau)
+
\sum_{j=1}^{N}
\exp(\mathrm{sim}(z_i, z_j^-)/\tau)
}
$$
这个公式看起来复杂，但意思很简单。

| 符号                                       | 通俗含义                               |
| ------------------------------------------ | -------------------------------------- |
| $z_i$                                      | 当前样本的表示，比如某个商品的图片表示 |
| $z_i^+$                                    | 正样本表示，比如同一个商品的文本表示   |
| $z_j^-$                                    | 负样本表示，比如其他商品的表示         |
| $\mathrm{sim}(\cdot)$                      | 相似度，常用余弦相似度                 |
| $\tau$                                     | 温度系数，控制区分程度                 |
| $\exp(\mathrm{sim}(z_i, z_i^+)/\tau)$      | 当前样本和正样本有多像                 |
| $\sum \exp(\mathrm{sim}(z_i, z_j^-)/\tau)$ | 当前样本和一堆负样本有多像             |

整个分数可以理解为：
$$
\frac{当前样本和正样本的相似度}{当前样本和正样本的相似度 + 当前样本和所有负样本的相似度}
$$
模型希望这个比例越大越好。

也就是说：

- 如果 $z_i$ 和 $z_i^+$ 很像，分子变大，loss 变小。
- 如果 $z_i$ 和负样本 $z_j^-$ 很像，分母变大，loss 变大。
- 所以模型会被迫学习：**正样本拉近，负样本推远。**

举个更直观的例子：

假设你在训练一个模型认识“同一个商品”。

商品 A 有图片、标题、评论：

- 图片：“一双白色运动鞋”
- 标题：“男款白色跑步鞋”
- 评论：“穿起来很轻便”

这些都描述同一个商品，所以它们应该在表示空间里靠得近。

而另一个商品 B 是“黑色背包”，商品 C 是“蓝牙耳机”，它们和商品 A 不是同一个东西，所以应该离 A 远一点。

放到多模态推荐里，对比学习经常这样用：
$$
\mathcal{L}_{cl}
=
-\log
\frac{
\exp(\mathrm{sim}(e_i^{img}, e_i^{txt})/\tau)
}{
\sum_{j}
\exp(\mathrm{sim}(e_i^{img}, e_j^{txt})/\tau)
}
$$
这里的意思是：

**第 $i$ 个商品的图像表示 $e_i^{img}$，应该和它自己的文本表示 $e_i^{txt}$ 更接近，而不是和其他商品的文本表示 $e_j^{txt}$ 更接近。**

所以，对比学习本质上不是直接问“这个用户会不会点击这个商品”，而是在训练模型学会更好的表示：

**哪些东西应该像，哪些东西不应该像。**

一句话总结：**对比学习就是通过“拉近正样本、推远负样本”，让模型学会更有区分度的表示。**
