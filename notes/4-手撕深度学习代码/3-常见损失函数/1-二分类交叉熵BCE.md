# 二分类交叉熵 BCE

比如判断一张图片是不是猫：

- `y = 1`：是猫
- `y = 0`：不是猫
- `p`：模型预测“是猫”的概率，比如 `0.8`

BCE 的核心思想非常简单：

> **预测越接近真实标签，Loss 越小；预测越离谱，Loss 越大。**

------

### BCE 公式

单个样本：
$$
\boxed{
L=-[y\log(p)+(1-y)\log(1-p)]
}
$$
其中：

- $y$：真实标签，只能是 `0` 或 `1`
- $p$：模型预测为正类的概率，范围是 `0~1`

这个公式看着复杂，其实分两种情况就很好理解。



### 代码实现

```python
import torch

def bce_loss(y, p):
    loss = -(y * torch.log(p) + (1 - y) * torch.log(1 - p))
    return loss.mean()
```
