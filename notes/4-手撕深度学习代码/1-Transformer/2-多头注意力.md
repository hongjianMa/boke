# 多头注意力机制

## 理解 Multi-Head Attention
如果只有一个 Attention：

```
吃 → 苹果
```

可能主要学到的是：

> 动宾关系

但是一句话里面其实同时存在很多种关系：

```
语义关系
语法关系
位置关系
上下文关系
长距离依赖
```

如果只让一个 Attention 去学习所有关系，能力有限。

所以提出：Multi-Head Attention

核心思想：

> 不要只让一组 Attention 学，而是让多组 Attention 并行学习不同的关系。

例如：

```
Head 1
吃 → 苹果

可能学习：
动宾关系


Head 2
我 → 喜欢

可能学习：
主谓关系


Head 3
吃 → 喜欢

可能学习：
上下文关系


Head 4
苹果 → 吃

可能学习：
语义依赖
```

> 不同注意力头可以在不同表示子空间中学习不同的依赖关系。

假设：

```
embedding dim D = 512

num_heads H = 8
```

那么：

```
head_dim = 512 / 8 = 64
```

原来的一个：

```
512维 Attention
```

拆成：

```
8 个 64维 Attention
```

也就是：

```
Head 1：64维
Head 2：64维
Head 3：64维
...
Head 8：64维
```

每个 Head：

```
Q K V
↓
Attention
↓
得到自己的输出
```

然后：

```
Head1
Head2
Head3
...
Head8

↓ concat

512维

↓ Linear

最终输出
```

所以 Multi-Head Attention 的流程一句话就是：

> **先生成 QKV，再拆成多个 Head，每个 Head 独立做 Attention，最后拼接所有 Head 并进行一次线性映射。**

## 代码实现

```python
import torch
import torch.nn as nn
import math


class MultiHeadAttention(nn.Module):

    def __init__(self, dim, num_heads):
        super().__init__()

        assert dim % num_heads == 0

        self.dim = dim
        self.num_heads = num_heads
        self.head_dim = dim // num_heads

        self.W_q = nn.Linear(dim, dim)
        self.W_k = nn.Linear(dim, dim)
        self.W_v = nn.Linear(dim, dim)

        self.W_o = nn.Linear(dim, dim)

    def forward(self, x):
        # x: (B, L, D)

        B, L, D = x.shape

        # 1. 计算 Q K V
        Q = self.W_q(x)
        K = self.W_k(x)
        V = self.W_v(x)

        # (B, L, D)
        # -> (B, L, H, head_dim)
        # -> (B, H, L, head_dim)

        Q = Q.view(B, L, self.num_heads, self.head_dim).transpose(1, 2)
        K = K.view(B, L, self.num_heads, self.head_dim).transpose(1, 2)
        V = V.view(B, L, self.num_heads, self.head_dim).transpose(1, 2)

        # 2. Attention score
        scores = Q @ K.transpose(-2, -1)

        scores = scores / math.sqrt(self.head_dim)

        # scores: (B, H, L, L)

        # 3. softmax
        attn = torch.softmax(scores, dim=-1)

        # 4. 加权 V
        output = attn @ V

        # output: (B, H, L, head_dim)

        # 5. 多头拼接
        output = output.transpose(1, 2).contiguous()

        # (B, L, H, head_dim)
        # -> (B, L, D)

        output = output.view(B, L, D)

        # 6. 最后线性映射
        output = self.W_o(output)

        return output
```

## 面试必备问题库

### Q1：什么是 Self-Attention？

可以回答：

> Self-Attention 是一种根据序列内部元素之间的相关性，对不同位置的信息进行动态加权聚合的机制。Q、K 和 V 都来自同一个输入序列，通过 Q 和 K 的点积得到注意力分数，再经过 Softmax 得到权重，最后对 V 进行加权求和。

核心：

```
Q K V 来自同一个序列
```

所以叫：

```
Self
```

------

### Q2：什么是 Multi-Head Attention？

回答：

> Multi-Head Attention 将输入映射到多个不同的表示子空间，在每个子空间中独立计算 Attention，最后将各个 Head 的结果拼接并经过线性映射，从而使模型能够从不同表示子空间学习不同的依赖关系。

关键字：

```
多个表示子空间

独立 Attention

Concat

Linear
```

------

### Q3：为什么要多头？

> 单个注意力头只能在一个表示子空间中计算注意力关系，而多头机制允许模型在不同子空间中并行学习不同类型的依赖关系，从而增强模型的表达能力。

------

### Q4：为什么 Q、K、V 要分别做线性变换？

也就是为什么：

```
W_q
W_k
W_v
```

不同。

回答：

> 因为 Q、K 和 V 扮演不同角色。Q 用于查询匹配，K 用于被匹配，V 用于提供最终聚合的信息。通过不同的可学习线性映射，可以让模型分别学习适合查询、匹配和信息表达的表示空间。

非常通俗地说：

```
Q：我要找谁

K：我是谁

V：我有什么信息给你
```

------

### Q5：为什么要除以 sqrt(d_k)？

高频题。

公式：
$$
\frac{QK^T}{\sqrt{d_k}}
$$
回答：

> 当向量维度较大时，Q 和 K 的点积方差会随维度增大，使注意力分数绝对值变大，Softmax 容易进入饱和区，导致梯度过小。因此除以 $\sqrt{d_k}$ 对点积结果进行缩放，提高训练稳定性。

简单版：

```
维度越大
↓
点积越大
↓
Softmax太尖锐
↓
梯度不好

所以除 sqrt(d_k)
```



------

### Q6：QK^T 的 Shape 是什么？

必须秒答。

如果：

```
Q:

(B,H,L,d)

K:

(B,H,L,d)
```

那么：

```
K^T:

(B,H,d,L)
```

所以：

```
QK^T:

(B,H,L,L)
```

------

### Q7：为什么 Multi-Head 后还要 W_o？

```python
output = self.W_o(output)
```

回答：

> 多个 Head 拼接后只是简单地将不同子空间的信息组合在一起，输出线性层 $W_O$ 可以进一步对不同 Head 的信息进行融合和重新映射。

也就是：

```
Concat：

只是拼起来

W_o：

真正进一步融合
```

------

### Q8：为什么 dim 必须能够被 num_heads 整除？

因为：

```
head_dim = dim / num_heads
```

例如：

```
dim = 512
heads = 8

每头 = 64
```

代码：

```
assert dim % num_heads == 0
```

因为通常每个 Head 保持相同维度。

------

### Q9：多头是不是参数量比单头多 H 倍？

不会。

> **多头注意力通常不会让参数量变成单头的 $H$ 倍，因为总维度 $D$ 不变，只是把 $D$ 拆成 $H$ 个 $d=D/H$ 维的头。**

例如：
$$
D=512,\quad H=8,\quad d=64
$$
8 个头总维度仍然是：
$$
8\times64=512
$$
因此 Q、K、V 的投影参数量仍约为 $3D^2$，**不是 $3HD^2$**。

### Q10：Self-Attention 和 Multi-Head Attention 什么关系？

Self-Attention 指 **Q、K、V 来自同一个输入序列**；Multi-Head Attention 指 **并行使用多个注意力头**。

两者可以结合，Transformer 中常用的是 **Multi-Head Self-Attention（多头自注意力）**。

---



### Q11：Self-Attention 和 Cross-Attention 区别？

Self-Attention：

```
Q = Wq(x)
K = Wk(x)
V = Wv(x)
```

都来自：

```
x
```

Cross-Attention：

```
Q：

来自一个序列

K V：

来自另一个序列
```

例如 Encoder-Decoder Transformer：

```
Q：

Decoder

K：

Encoder

V：

Encoder
```

核心区别：

> Self-Attention 的 Q、K、V 来自同一序列，而 Cross-Attention 的 Q 与 K/V 来自不同序列。

------

### Q12：Multi-Head Attention 和普通 Attention 最大区别？

普通单头：

```
整个 D 维空间

做一次 Attention
```

多头：

```
拆成多个子空间

每个子空间分别 Attention

最后拼接
```
