# 自注意力
注意力机制标准公式：
$$
\boxed{
\operatorname{Attention}(Q,K,V)
=
\operatorname{Softmax}
\left(
\frac{QK^T}{\sqrt{d_k}}
\right)V
}
$$
其中：

- $Q$：Query，查询
- $K$：Key，键
- $V$：Value，值
- $d_k$：$Q,K$ 的特征维度

可以简单记成：
$$
\boxed{QK^T \rightarrow 除以\sqrt{d_k}\rightarrow Softmax\rightarrow \times V}
$$

code实现
```python
import torch
import torch.nn as nn
import math

class SelfAttention(nn.Module):
    def __init__(self, dim):
        super().__init__()

        self.W_q = nn.Linear(dim, dim)
        self.W_k = nn.Linear(dim, dim)
        self.W_v = nn.Linear(dim, dim)

    def forward(self, x):
        # x: (batch, seq_len, dim)

        Q = self.W_q(x)
        K = self.W_k(x)
        V = self.W_v(x)

        scores = Q @ K.transpose(-2, -1)
        scores = scores / math.sqrt(Q.size(-1))

        attn = torch.softmax(scores, dim=-1)

        output = attn @ V

        return output, attn
